# What is ForesightJS

<WhatIsIntro />

In Vue you use ForesightJS through the official `@foresightjs/vue` package: the `v-foresight` directive registers your elements, and composables expose their reactive prediction state.

## Installation

```bash
npm install @foresightjs/vue
```

Continue with [Installation](./installation.md) and the [Quick Start](./quick-start.md).
